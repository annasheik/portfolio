Plan Your Day App

An application to find out the weather and the best venues around you in any city in the world. The website pulls data from the third-party API's: OpenWeatherMap.org and Foursquare.com, and uses Google Places API to verify the location.

The user is able to find out the current weather in any location in the world and also search for venues around, sorted by categories. The user is provided with the photos for each venue, its address and the url to the venue's website. Fully responsive
